# Script Runner Wiki

Welcome to the Script Runner wiki! This documentation will help you understand and use Script Runner effectively.

## Overview

Script Runner is a web-based application that allows you to manage, schedule, and execute Python and Bash scripts through an intuitive interface. Whether you're automating tasks, running scheduled jobs, or managing multiple scripts, Script Runner provides the tools you need.

## Key Features

- Create and manage Python and Bash scripts
- Schedule script executions using cron syntax
- Install script dependencies automatically
- View execution history and logs
- Tag-based organization
- Real-time execution monitoring
- Discord notifications integration
