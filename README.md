# Script Runner

Script Runner is a web-based application that allows users to manage, schedule, and run Python and Bash scripts. It provides a user-friendly interface for creating, editing, and executing scripts, as well as managing their requirements and schedules.

## Features

- Create and manage Python and Bash scripts
- Schedule script execution using cron expressions
- Install and manage script requirements
- View execution logs and output
- Tag-based organization of scripts
- Responsive web interface

## Technologies Used

- Next.js 13 with App Router
- React
- TypeScript
- Tailwind CSS
- Docker
- Python 3
