# Script Runner

## 🚧🚨⚠️UNDER HEAVY DEVELOPMENT⚠️🚨🚧

Script Runner is a powerful web-based application designed for managing, scheduling, and executing Python and Bash scripts. It provides a user-friendly interface that simplifies script management, offers detailed execution history, and streamlines dependency installation.

## Features

- **Script Management**: Create, edit, and organize Python and Bash scripts through an intuitive web interface.
- **Scheduling**: Set up automated script executions using cron syntax for flexible scheduling.
- **Dependency Management**: Easily install and manage script dependencies directly from the interface.
- **Execution History**: View comprehensive logs and history of all script executions.
- **Tagging System**: Organize scripts efficiently with a tag-based system for easy categorization and retrieval.
- **Real-time Monitoring**: Monitor script executions in real-time with detailed output and error reporting.
- **User-friendly Interface**: Navigate through the application with a clean, responsive design optimized for desktop and mobile devices.
